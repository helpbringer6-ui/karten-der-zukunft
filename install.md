# Installation

## Voraussetzungen
- Node.js und npm installiert (Version 14 oder höher empfohlen)
- Zugriff auf ein ARKit oder ARCore fähiges Gerät

## Schritte zur Installation

1. Klone das Repository:
   ```bash
   git clone https://github.com/deinbenutzername/karten-der-zukunft.git
   cd karten-der-zukunft
   ```

2. Installiere die notwendigen Abhängigkeiten:
   ```bash
   npm install
   ```

3. Konfiguriere die AR-Umgebung, indem du die Anweisungen in der `config.md` befolgst. Stelle sicher, dass alle erforderlichen Einstellungen korrekt vorgenommen werden.
