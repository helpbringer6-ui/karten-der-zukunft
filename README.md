# Karten der Zukunft

## Projektbeschreibung
**Karten der Zukunft** ist ein Augmented-Reality-Projekt, das es Benutzern ermöglicht, historische Karten und deren ursprüngliche geografische Standorte zu erkunden. Durch die Verwendung eines AR-Headsets oder einer mobilen App können Benutzer in die Vergangenheit reisen und die Veränderungen in der Landschaft und den urbanen Strukturen im Laufe der Zeit visuell erleben. 

## Funktionen
- AR-Überlagerungen von historischen Karten
- Interaktive Zeitleisten zur Auswahl verschiedener Zeitperioden
- Geolokalisierung zur Darstellung von Karten an ihrem ursprünglichen Standort
- Informationsschichten mit Details zu historischen Ereignissen und geografischen Veränderungen

## Technologien
- Augmented Reality (AR) Framework: ARKit / ARCore
- Programmiersprachen: Swift / Kotlin
- 3D-Modellierung für historische Gebäude
- Nutzung von GIS-Daten

## Installation
1. Klonen Sie das Repository.
2. Installieren Sie die erforderlichen Abhängigkeiten.
3. Konfigurieren Sie die AR-Umgebung nach den Anweisungen in der `install.md`.

## Nutzung
- Starten Sie die App.
- Wählen Sie einen Standort in der Karte aus.
- Wählen Sie eine historische Karte aus, die über der aktuellen Landschaft angezeigt werden soll.
